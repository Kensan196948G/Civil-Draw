# 1. プロジェクト概要

## 1.1 背景と目的

建設土木業における現場作業では、仮設計画・土工計画・施工ヤード配置・道路舗装計画等の平面図を頻繁に作成・更新する必要がある。従来はAutoCADをはじめとする高価な商用CADソフトウェアに依存してきたが、ライセンスコスト・導入負荷・操作習熟の障壁が課題となっている。

本プロジェクトは、建設土木業に特化したWebベースの2D CADツールをOSSライブラリにより内製開発し、社内9拠点への展開を実現することで、コスト削減・業務効率化・図面品質の標準化を達成することを目的とする。

# 2. 機能要件

## 2.1 キャンバス・操作系

| **機能ID** | **機能名**   | **詳細仕様**                                                                | **優先度** |
|------------|--------------|-----------------------------------------------------------------------------|------------|
| CV-001     | ズーム操作   | マウスホイールによるズームイン/アウト（0.1x〜50x）。Ctrl+スクロールも対応。 | 必須       |
| CV-002     | パン操作     | Spaceキー+ドラッグ または 中ボタンドラッグによるキャンバス移動              | 必須       |
| CV-003     | グリッド表示 | 実寸メートル単位のグリッド。縮尺に連動して間隔を自動調整                    | 必須       |
| CV-004     | スナップ機能 | グリッドスナップ・端点スナップ・中点スナップ・交点スナップ                  | 必須       |
| CV-005     | 座標表示     | ステータスバーに現在カーソル座標をリアルタイム表示（実寸m単位）             | 必須       |
| CV-006     | 縮尺設定     | 1/50, 1/100, 1/200, 1/500, 1/1000 から選択                                  | 必須       |
| CV-007     | 用紙設定     | A4/A3/A2/A1/A0（縦・横）の用紙サイズ選択・用紙枠表示                        | 必須       |

## 2.2 描画ツール

| **機能ID** | **機能名**         | **詳細仕様**                                                   | **優先度** |
|------------|--------------------|----------------------------------------------------------------|------------|
| DT-001     | 選択・移動         | 図形の選択（単体/矩形範囲選択）、移動、コピー、削除            | 必須       |
| DT-002     | 線分ツール         | 単線・連続線の描画。入力座標値での確定対応                     | 必須       |
| DT-003     | 矩形ツール         | 2点指定による矩形描画。回転対応                                | 必須       |
| DT-004     | 円・円弧ツール     | 中心+半径指定、3点指定による円描画。円弧（始点・終点・半径）   | 必須       |
| DT-005     | ポリラインツール   | 複数頂点を持つ折れ線。閉合ポリゴン対応                         | 必須       |
| DT-006     | テキストツール     | 任意位置への文字配置。日本語対応・フォントサイズ・回転角度設定 | 必須       |
| DT-007     | 寸法線ツール       | 水平・垂直・平行・角度寸法。寸法スタイル設定（矢印・文字高）   | 必須       |
| DT-008     | ハッチングツール   | 土工・舗装断面等のハッチングパターン。建設業標準パターン対応   | 推奨       |
| DT-009     | シンボルライブラリ | 建設土木専用シンボル（仮囲い・重機・コーン・測量杭等）の配置   | 推奨       |
| DT-010     | 元に戻す/やり直し  | Ctrl+Z/Ctrl+Y による操作履歴管理（最大100ステップ）            | 必須       |

# 3. 非機能要件

| **カテゴリ**   | **要件項目**     | **目標値・基準**                                              |

| 互換性         | ブラウザ対応     | Chrome/Edge 最新2バージョン                                   |

# 6. 制約・前提条件

## 6.1 技術的制約

- DWG形式の直接出力は不可。DXF出力後にODA File Converter等で変換する運用とする

- ブラウザのCanvas APIを使用するため、印刷解像度はモニター依存（高精細印刷はPDF経由を推奨）

- オフライン動作を前提とするため、クラウドストレージ連携はPhase 3以降の検討事項とする

## 6.2 運用前提

- 対象ブラウザ：Microsoft Edge（社内標準）・Google Chrome

- Entra ID参加済みPC（SKYSEA Client View管理下）での利用を前提とする

- ISO 27001・J-SOX準拠のため、図面データの外部送信は禁止。ローカル処理のみとする

- 社内展開時はITシステム運用管理部が展開・管理を担当する
